# Tirex

## build instructions
```
  apt install libipc-sharelite-perl libjson-perl libgd-gd2-perl libwww-perl devscripts
  git clone https://github.com/openstreetmap/tirex.git
  tirex/
  make
  make deb
  make install
  make install-example-map
  make install-munin
  make install-nagios  
```
